# Elegant Furniture Hub (Royal Furniture)

A premium, enterprise-grade e-commerce interface for a luxury furniture brand, built with a focus on "Luxe Minimalism." This project transforms a standard layout into a highly polished, responsive, and interactive frontend experience.

## ✨ Key Features
- **Luxe Minimalist Design**: A sophisticated aesthetic using deep charcoal, muted golds, and elegant typography (Playfair Display & Inter).
- **Responsive Navigation**: Glassmorphic effects, sticky headers, and smooth mobile menus.
- **Dynamic Cart System**: Add to cart, calculate totals, and remove items efficiently using pure JavaScript and LocalStorage.
- **Client-Side Authentication**: Secure, split-screen login/signup flows with comprehensive form validation and LocalStorage user persistence.
- **Admin Dashboard**: A dedicated administrative portal with interactive data tables, simulated revenue tracking, and Product CRUD capabilities.
- **Refined UX**: Fade-in scroll animations via `IntersectionObserver`, sleek Toast notifications replacing native browser alerts, and an elegant dark mode toggle.

## 🛠️ Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Custom Variables, Flexbox/Grid, Glassmorphism, Animations)
- Vanilla JavaScript (DOM manipulation, LocalStorage, Custom Interactions)
- Bootstrap 5 (Grids, Modals, Toasts)
- Boxicons & Font Awesome (Iconography)
- Google Fonts

## 🚀 Getting Started
To view this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdurrehman510/Elegant-Furniture-Hub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Elegant-Furniture-Hub
   ```
3. Open `index.html` in your favorite browser, or start a local development server:
   ```bash
   # Using Python 3
   python -m http.server 8080
   
   # Using Node.js (http-server)
   npx http-server .
   ```
